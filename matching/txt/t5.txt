<p>Muhurtha Sindhu: Mitra-Mitra = 5 ; Mitra-Sama = 4 ; Sama-Sama = 3 ;Mitra-Satru= 2;  Sama-Satru = 1 ; Satru-Satru = 0 - points. </p>

<p><span class = "minahaa">Exceptions:</span><br>
<b>Muhurtha Sindhu</b>: If this kootami exists, varna, gana, yoni, bha, dina - kootamis are taken care of. If this koatami does not exist, then freindship between chandra navamsas' lords of the girl and boy will do.<br>
<b>BV Raman</b>: If this koatami does not exist, then if freindship between chandra navamsas' lords of the girl and boy, exists, then the match is definitely acceptable.<br>
